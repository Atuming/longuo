# 实现计划：小说辅助编辑器（Novel Assistant Editor）

## 概述

基于 React + TypeScript + CodeMirror 6 构建纯客户端小说辅助编辑器。数据以 `.novel` JSON 文件存储在本地，通过 File System Access API 读写。AI 辅助写作通过浏览器直接调用外部 AI 模型 API 实现，支持多提供商配置和自定义 Prompt 模板。实现按照数据层 → 状态层 → UI 层的顺序推进，确保每一步都可增量验证。
